## Overview
VisionBoard is a full-stack web application built to help students track and manage co-op and internship applications. The platform provides a Kanban-style workflow, real-time updates, and secure user authentication to streamline the application process.

This project demonstrates frontend development, real-time databases, authentication, and API integration.

---

## Features
- Kanban board for application status tracking  
- Filters and deadlines for improved organization  
- Secure authentication using Google Auth  
- Real-time CRUD operations with Firebase Firestore  
- Per-user workspaces with access control  
- Integration with public job listing APIs  

---

## Tech Stack
- JavaScript  
- React  
- Firebase Firestore  
- Google Authentication  
- REST APIs  

---

## How It Works
1. Users authenticate using Google Auth.
2. Each user is assigned a secure, isolated workspace.
3. Applications can be created, updated, and deleted in real time.
4. Firestore security rules enforce per-user data access.
5. Applications are displayed in a Kanban-style interface.
